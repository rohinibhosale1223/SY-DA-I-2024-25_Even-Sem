BMW M5

The BMW M5 is a high-performance luxury sedan that blends comfort with exhilarating power. It features BMW's M xDrive system and a potent twin-turbo V8 engine.

Key Specs:

  • Engine: 4.4L Twin-Turbo V8
  • Horsepower: 600-617 HP
  • 0-60 mph: ~3.1 seconds
  • Top Speed: 190 mph
